# Web Scraper

This project contains a simple Python web scraper that extracts question titles and descriptions from the Stack Overflow Python tag page and saves them to a CSV file.

## How to Run

1. Install dependencies: `pip install -r requirements.txt`
2. Run the scraper: `python main.py`

The script will scrape the latest questions, save them to `qustions.csv`, and send the CSV via email.
